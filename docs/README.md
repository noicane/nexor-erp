## 📦 Paket İçeriği

Bu paket, mevcut AtmoLogic ERP sisteminizi **Redline Nexor** kurumsal kimliğine dönüştürmek için gereken tüm dosyaları içerir.

### Klasör Yapısı

```
nexor_rebrand_complete/
├── assets/                  # Logo ve icon dosyaları (6 adet)
│   ├── logo_web.png        # 800x533 - Web/Marketing
│   ├── logo_sidebar.png    # 400x267 - Sidebar
│   ├── logo_login.png      # 320x213 - Login ekranı
│   ├── logo_small.png      # 200x133 - Header
│   ├── icon.ico            # 64x64 - Favicon (ICO)
│   └── favicon.png         # 64x64 - Favicon (PNG)
│
├── themes/
│   └── nexor_theme.py      # Redline Nexor tema sistemi
│
├── dialogs/
│   └── nexor_login.py      # Premium login ekranı
│
├── docs/
│   └── REBRANDING_GUIDE.md # Detaylı rebrand rehberi
│
├── install_nexor.py         # Otomatik kurulum scripti
├── QUICK_START.md           # Hızlı başlangıç rehberi
└── README.md                # Bu dosya
```

---

## 🚀 Hızlı Başlangıç

### Adım 1: Otomatik Kurulum

```bash
python install_nexor.py
```

Kurulum scripti:
1. Hedef ERP klasörünü sorar
2. Dosyaları otomatik kopyalar
3. Yedekleme oluşturur
4. Konfigürasyonu günceller

### Adım 2: Manuel Kurulum (İsteğe Bağlı)

Detaylı talimatlar için `QUICK_START.md` dosyasına bakın.
